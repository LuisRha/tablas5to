Codigo - tabla - html:

"Las tablas en HTML y CSS son elementos fundamentales para organizar y presentar datos de manera estructurada en una página web. Permiten mostrar información de forma ordenada en filas y columnas, lo que facilita la comprensión y la visualización de datos. En HTML, las tablas se crean utilizando elementos como < table>, < tr>, < th> y < td>. El elemento < table> define la estructura de la tabla, < tr > representa una fila, < th> se utiliza para las celdas de encabezado y < td> para las celdas de datos. Por otro lado, CSS se utiliza para estilizar y personalizar el aspecto de las tablas. Se pueden aplicar estilos como colores de fondo, bordes, márgenes y alineación de texto para mejorar la apariencia de la tabla y hacerla más atractiva visualmente. Además, CSS ofrece la posibilidad de hacer que las tablas sean responsivas, es decir, que se adapten automáticamente al tamaño de la pantalla del dispositivo en el que se visualizan. Esto se logra mediante el uso de técnicas como media queries y unidades de medida relativas, que permiten ajustar el diseño de la tabla según el ancho de la pantalla. En resumen, las tablas en HTML y CSS son herramientas poderosas para presentar datos de manera efectiva en una página web, ofreciendo flexibilidad y control sobre su apariencia y comportamiento."
